No. Topic Description
1 ความแตกต่างของ Property และ Value
  • Property สำหรับกำหนดส่วนของคุณสมบัติที่ต้องการให้แสดง
  • Value สำหรับกำหนดค่าที่ต้องการให้แสดง
2 Property มี 5 กลุ่ม
  1. Property for content: Color, Size, Font, Text, List-Style-Type, Height, Width, Background
  2. Property for border: Border, Table
  3. Property for padding: Padding (top, right, buttom, left)
  4. Property for margin: Margin (top, right, buttom, left)
  5. Property for element: Display, Height, Width, Position, Float, Align, Flex, Grid
3 Value Unit มี 2 ประเภท
  1. Absolute lenghts
    • cm เซนติเมตร
    • mm มิลลิเมตร
    • in นิ้ว
    • px พิกเซล 1/96 นิ้ว
    • pt พอยท์ 1/72 นิ้ว
    • pc พิคาล 1/12 นิ้ว
  2. Relative lenghts
    • em เท่าขนาดฟอนต์ของ element
    • ex เท่าความสูงของฟอนต์ของ element
    • ch ความกว้างของตัวหนังสือใน parent element
    • rem เท่าขนาดฟอนต์ของ root element
    • vh เปอร์เซ็นต์ความสูงหน้าจอ
    • vw เปอร์เซ็นต์ความกว้างหน้าจอ
    • vmin เปอร์เซ็นต์ของ view point ต่ำสุด
    • vmax เปอร์เซ็นต์ของ view point สูงสุด
    • % เปอร์เซ็นต์ของ parent element

2 ความแตกต่างของ Grid และ Flex
  • Flex ใช้เพื่อจัดการ layout 1D ส่วน Grid ใช้เพื่อจัดการ layout 2D
    • Flex จะสามารถจัดวางของต่างๆภายใน layout 1 แถว อาจจะเป็นแนวตั้งหรือแนวนอน
    • Grid จะสามารถจัดของไว้ตรงไหนก็ได้ใน grid layout
3 ความแตกต่างของ Justify and Align
  • Justify จะใช้กับ flex และ grid โดยจะจัดการในแนวนอน ส่วน Align จะใช้กับ flex โดยจะจัดการในแนวตั้ง
4 Position
  • การจัดตำแหน่ง ประกอบด้วย
    • Static : เป็นค่าพื้นฐานของทุกบล็อกโดนจะเลื่อนตามหน้าเว็บปกติ
    • Relative : การเลื่อนออกจากตำแหน่งเดิม ตามที่เรากำหนดการดัน block จากทาง ซ้าย ขวา บน ล่าง
    • Fixed : กำหนดให้ block นั้นยึดอยู่กับที่บนหน้าเว็บ
    • Alsolute : การเลื่อนออกจากมุมซ้ายบนสุดของหน้าจะ ตามการกำหนดการดัน block จากทาง ซ็าย ขวา บน ล่าง
    • Sticky : การรวม relative กับ fixed ไว้ด้วยกัน โดยการกำหนด relative เพิ่มว่าจะให้ไป fix ที่ top หรือ bottom ทำให้เราเห็น block นั้นเลื่อนและถูกตรีงไว้เมื่อเรา เลื่อน scroll เมาส์
5 Z-index
  • เป็นการจัดการ element ในรูปแบบชั้น layer บนหน้าจอ โดยค่าน้อยจะอยู่ด้านหลัง ส่วนค่ามากจะอยู่ด้านหน้า ค่าเริ่มต้นของทุก element คือ z-index = 0
